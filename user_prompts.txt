────────────────────────────────────────────────────────────────────────
USER PROMPT COMBINATIONS — Agentic Cold Emailer System
────────────────────────────────────────────────────────────────────────

NOTE:
- Emails are generated PER (Company, Hiring Role) group.
  e.g. 8 HR records with 8 unique company+role combos → 8 tailored emails.
  Each email mentions the specific company name, role, and uses the
  correct greeting (person name → "Dear Priya,", team → "Dear Hiring Team,").
- A single chat interface handles both email requests and general resume Q&A.
- Filters are extracted from the prompt using LLM-based intent parsing.
- Progressive fallback: if exact filters yield no matches, the system
  relaxes filters (drop role → drop company type → all contacts).
- NO links are included in any email (they cause bounces).

════════════════════════════════════════════════════════════════════════
K) EDGE CASES
════════════════════════════════════════════════════════════════════════
40. draft me a cold email for AI Engineer position in ABC company
    → if ABC has no AI Engineer opening → fallback: emails for all roles at ABC

41. generate email for a role that doesn't exist in the CSV
    → fallback: relaxes role filter, matches on other filters or all contacts

42. typos / variations (handled by LLM fuzzy parsing):
    - "startup compaies" → Startup
    - "product base" → Product Based
    - "service based" → IT Services & Consultancy
    - "python dev" → Python Developer

43. ambiguous intent:
    - "tell me about cold emails" → is_email_request=False (general question)
    - "can you draft something for startups?" → is_email_request=True

44. follow-up in chat:
    - "now do the same for fintech companies" → new email request

45. same company + same role but multiple HR contacts:
    → generates ONE email for that group, listed recipients under it
    e.g. 3 HR people at ABC all hiring AI Engineer → 1 email, 3 recipients

46. greeting edge cases:
    - HR Name = "Priya" → "Dear Priya,"
    - HR Name = "HR Team" → "Dear Hiring Team,"
    - HR Name = "saisrushik" → "Dear Saisrushik,"
    - Multiple HRs in same group → "Dear Hiring Team,"
