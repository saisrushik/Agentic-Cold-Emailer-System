────────────────────────────────────────────────────────────────────────
USER PROMPT COMBINATIONS — Agentic Cold Emailer System
────────────────────────────────────────────────────────────────────────

NOTE:
- The system generates ONE email per query and sends it to ALL matched HR/TA contacts.
- The chat interface handles both email requests and general resume Q&A.
- Filters are extracted from the prompt using LLM-based intent parsing.
- Progressive fallback: if exact filters yield no matches, the system
  relaxes filters (drop role first, then company type) before falling
  back to all contacts.
